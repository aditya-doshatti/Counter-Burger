## Users

```
Database used - MySQL
Table Name - user
CREATE TABLE user (
    userid varchar(255) NOT NULL,
    fname varchar(255) NOT NULL,
    lname varchar(255),
    email varchar(255) NOT NULL,
    password varchar(255) NOT NULL,
    PRIMARY KEY (email)
);
```
